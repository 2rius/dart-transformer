# Changelog

All notable changes to `dart-transformer` will be documented in this file.

## 0.1.2 - 2026-02-24

### What's Changed

* Added support for Optional in union types when transforming PHP Data properties to Dart types.
* Improved handling of nullable and PHPDoc-annotated array types (@var Type[] and @var array<Type>).
* Updated DataClassTransformer tests to cover combinations of Optional, nullable, enum, and array types.

**Full Changelog**: https://github.com/2rius/dart-transformer/compare/0.1.1...0.1.2
